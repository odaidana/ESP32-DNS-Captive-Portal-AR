# دليل بناء سيرفر DNS والصفحة الترحيبية باستخدام ESP32

يقدم هذا المستودع نموذجاً تعليمياً حقيقياً ومفتوح المصدر لكيفية بناء سيرفر DNS محلي وصفحة ترحيبية (Captive Portal) باستخدام معالج الـ ESP32. يوضح المشروع كيفية اعتراض طلبات الـ DNS وتوجيه الأجهزة المتصلة إلى صفحة ويب مستضافة بالكامل داخل المعالج.

## التقنيات المستخدمة
- **نقطة وصول محلية (SoftAP):** إعداد شبكة واي فاي مستقلة وخاصة بالبوردة.
- **اعتراض الـ DNS:** تشغيل خادم DNS على المنفذ (Port 53) لتحويل أي رابط يكتبه المستخدم (`*`) إلى عنوان الـ IP الخاص بالبوردة.
- **خادم الويب (Web Server):** عرض صفحات HTML مخصصة للمستخدمين فور اتصالهم بالشبكة.

## هيكلة الملفات
- `src/dns_portal.cpp` — الكود البرمجي الأساسي لتشغيل الشبكة وسيرفر الـ DNS مع تعليقات توضيحية باللغة العربية.
